# Songmont 大地有道理 Case Study Website

这是一个可直接上传 GitHub Pages 的静态网站包。

## 使用方法
1. 解压 zip 文件
2. 将 `index.html` 和 `assets/` 文件夹上传到 GitHub 仓库根目录
3. 在 GitHub 仓库设置中开启 GitHub Pages
4. Source 选择 `Deploy from a branch`
5. Branch 选择 `main` / root
6. 保存后等待生成链接

## 本地预览
直接双击 `index.html` 即可打开。

## 修改内容
- 修改文字：编辑 `index.html`
- 修改样式：编辑 `assets/css/style.css`
- 替换图片：替换 `assets/images/` 中对应图片文件，保持文件名一致即可。
